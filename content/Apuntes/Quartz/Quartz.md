[Quartz](https://quartz.jzhao.xyz) es un generador de estilos estáticos, creado específicamente para convertir archivos de [[Markdown]] en un sitio web. 

Esta escrita con Node.js y TypeScript por lo que modificar y personalizar tu propio jardín es muy sencillo.

Es la herramienta por excelencia para la generación de un jardín digital, la forma mas simple de convertir tu Vault local de [[Obsidian]] en una pagina web, con la que puedes interactuar en cualquier momento.

# ¿Como usarlo?
---
En la pagina principal de [Quartz](https://quartz.jzhao.xyz) se explican los requerimientos mínimos necesarios para poder utilizar quartz, y se explica como empezar a usarlo.

Básicamente lo que se debe de hacer es clonar el repositorio donde se encuentra el proyecto original de quartz. Después puedes inicializar tu propio contenido de Quartz y guardarlo en tu propio repositorio.

# Build local
---
Antes de empujar cualquier cambio en tu repo es mas recomendado crear un **build** local y verificar que los cambios que deseas son correctos, para hacer esto ejecuta el comando:

``` bash
npx quartz build --serve 
```
