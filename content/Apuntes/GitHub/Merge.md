En ocasiones el contenido de tu repositorio local será muy diferente que el de tu repositorio remoto, normalmente esto ocurre cuando no has utilizado el repositorio local el mucho tiempo y hay cambios importantes que necesitas para estar actualizado con el remoto.
En estos casos se utiliza el Merge
